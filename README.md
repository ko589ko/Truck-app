# トラック配車管理アプリ

ドライバーと会社の間で、配車指示の共有とチャット連絡ができるアプリです。
Fly.io と PostgreSQL を使用し、ブラウザ上で動作します。

## 使用技術
- Node.js (Express)
- PostgreSQL (Fly.io)
- HTML / CSS / JavaScript

## 機能
1. ドライバーの新規登録、一覧表示、削除
2. 今日・明日の配車指示の確認
3. 過去の履歴表示
4. チャット機能によるメッセージの送受信

## 利用の流れ
1. 会社がドライバーを登録する
2. ドライバーは自身を選択して画面に入る
3. ドライバーはその日の指示を確認する
4. 必要に応じてチャットで連絡する
